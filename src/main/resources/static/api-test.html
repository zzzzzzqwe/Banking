Admin — Users

Эти запросы вернут 403 Forbidden если у текущего пользователя роль USER. Нужна роль ADMIN.

GET /api/admin/users ADMIN показать JSON

—

GET /api/admin/users/:id ADMIN

Кликни на UUID в таблице выше — поле заполнится автоматически.

—

PUT /api/admin/users/:id/deactivate ADMIN

Деактивированный пользователь не сможет войти в систему. Действие необратимо через UI.
